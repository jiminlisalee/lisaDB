# 데이터 웹사이트 만들기

## 개발환경
- MariaDB
- PHP
- HTML
- 리눅스 Apache Web Server

선택이유 : 서버가 분리되어있는 곳에서 개발을 하는 것이 더 안전하다 판단했으며 

## 데이터
- kaggle에서 데이터 가져오기

## 데이터 분석 기능 4가지
1. 한국 tvshow,movie를 방송년도 오름차순으로 확인 가능
2. 제목을 기입하여 해당 프로그램의 정보 확인 가능
3. 국가명을 기입하여 해당 국가의 프로그램을 확인 가능
4. 국가,방송년도 두 정보를 기입 후 조회 가능
